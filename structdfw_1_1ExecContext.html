Detailed Description

Runtime state for a cyclic executive instance.

This structure holds all configuration, buffers, and timing state needed by the executor. The caller must provide and own all pointed-to memory (devices, schedules, health, states) for the lifetime of the context. No dynamic allocation occurs within the executor.
